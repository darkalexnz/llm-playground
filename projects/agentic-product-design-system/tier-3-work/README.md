# Tier 3 — Working Files
*Document type: Index | Owner: Orchestrator | Tier: 3*
*Status: Active | Last updated: 15 March 2026 | Version: 1.0*

---

Ephemeral task outputs, working files, and brainstorm artifacts live here. Files in this folder are session-scoped by default.

## What belongs here

- Task outputs that span sessions and inform a Tier 2 document but aren't ready to merge
- Working drafts under active iteration
- Structured outputs awaiting review before promotion to Tier 2

## What does not belong here

- Final content — promote to the appropriate Tier 2 document instead
- Scratchpads and superseded drafts — discard at session close (chat-only)

## Naming convention

`YYYY-MM-DD-[task-slug].md`

Example: `2026-03-15-offer-editor-wireframe-draft.md`

## Archive

When a sprint or phase closes, move completed Tier 3 files to `/archive/[YYYY-MM-DD]-[label]/` before clearing this folder.

Full persistence policy: `orchestrator/session-protocol.md` section 5.
